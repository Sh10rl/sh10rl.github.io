## login

如果缺少[OpenSSL 1.1](https://mirror.cs.uchicago.edu/ubuntu/ubuntu/pool/main/o/openssl)可以下一个:

```bash
wget https://mirror.cs.uchicago.edu/ubuntu/ubuntu/pool/main/o/openssl/libssl1.1_1.1.1f-1ubuntu2.24_amd64.deb
sudo dpkg -i libssl1.1_1.1.1f-1ubuntu2.24_amd64.deb
```

`client`有一个读config：

```c
if (mode == 0) {
    memcpy(out, buf + 4, 16);       // bytes [4:19] -> key
} else if (mode == 1) {
    memcpy(out, buf + 23, 16);      // bytes [23:38] -> iv
} else if (mode == 2) {
    memcpy(out, buf + 46, 42);      // bytes [46:87] -> passwd
} else {
    free(buf);
    fclose(f);
    return 0;
}
```

传明文时除了"req login ..."都用RC4，传Key IV时用RSA。
server的state有0123分别对应：接收"req login ..."明文，接收rsa(IV)，接收rsa(Key)，接收aes-cbc(passwd)。每个state都有一个简单的for循环校验，不通过输出backdoor。
